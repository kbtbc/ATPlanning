## Audit Process

For each resupply location:
1. **Google Search Verification:**
   - Search business name + location
   - Verify phone numbers are current
   - Check if business is still operating
   - Update hours if changed
   - Verify website URLs
   - Check Google Maps coordinates

2. **Data Quality Checks:**
   - Remove permanently closed businesses
   - Update phone numbers if changed
   - Correct addresses if incorrect
   - Update hours if changed
   - Verify pricing is current (note if outdated)
   - Update websites if changed/broken

3. **Discrepancy Resolution:**
   - If conflicting info found, ask user for clarification
   - Document source of updated information
   - Note any significant changes in business notes

4. **Documentation:**
   - Mark location as audited in checklist below
   - Update changelog with changes made
   - Note any businesses removed or flagged
